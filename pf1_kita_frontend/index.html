Curriculum

Learn how to develop applications using object-oriented programming language Java and script programming language PHP
Learn the process of creating screens using html5/css/JS
Learn how to store and manage data using relational DBMS such as Oracle and MySQL and SQL, and learn DB management skills with programming languages such as Java/PHP.
Based on UI design output and GUI standards, students learn programming skills to design and implement UI/UX. In particular, students focus on 2D and 3D UI implementation techniques using Unity 3D engine.
Learn how to develop fast web applications using open source frameworks such as Spring Framework and Laravel Framework
Learn the skills to build and deploy applications in a serverless cloud environment.
By using the skills acquired during the training process, trainees conduct practical projects on their own.
